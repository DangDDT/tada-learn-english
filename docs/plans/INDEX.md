## Naming Convention

```
YYYY-MM-DD-<type>-<description>.md
```

Types:
- `sprint-<n>-log` — Sprint daily/weekly log
- `plan-<feature>` — Implementation plan for a feature
- `retro-<sprint>` — Sprint retrospective
- `decision-<topic>` — Architecture/design decision record
- `bug-<summary>` — Bug investigation log
